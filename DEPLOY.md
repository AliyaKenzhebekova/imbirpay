# Публикация ImbirPay через Google Apps Script

Бесплатно. Ссылка для всей команды без регистрации.

---

## Шаг 1 — Открыть Google Apps Script

Перейдите: https://script.google.com

Войдите через ваш Google-аккаунт.

---

## Шаг 2 — Создать проект

1. Нажмите **«+ Новый проект»**
2. Вверху переименуйте проект: нажмите на «Без названия» → введите **ImbirPay**

---

## Шаг 3 — Вставить Code.gs

В редакторе уже открыт файл `Code.gs`.
Удалите всё содержимое и вставьте:

```javascript
function doGet() {
  return HtmlService
    .createHtmlOutputFromFile('index')
    .setTitle('ImbirPay — Казначейство Imbir Group')
    .setXFrameOptionsMode(HtmlService.XFrameOptionsMode.ALLOWALL)
    .addMetaTag('viewport', 'width=device-width, initial-scale=1.0');
}
```

---

## Шаг 4 — Добавить файл index.html

1. Слева рядом с «Файлы» нажмите **«+»** → выберите **HTML**
2. Введите имя: **index** (без .html — оно добавится само)
3. Откроется пустой файл index.html
4. **Удалите всё содержимое**
5. Откройте файл: `/Users/admin/Desktop/Project/ImbirPay/index.html`
6. Выделите всё (Cmd+A) → скопируйте (Cmd+C)
7. Вставьте в редактор Apps Script (Cmd+V)

---

## Шаг 5 — Опубликовать

1. Нажмите кнопку **«Развернуть»** (правый верхний угол) → **«Новое развертывание»**
2. Нажмите на ⚙️ шестерёнку → выберите **«Веб-приложение»**
3. Заполните:
   - **Описание:** ImbirPay v1
   - **Выполнять как:** Я (ваш аккаунт)
   - **Кто имеет доступ:** ⭐ **Все** (анонимный доступ)
4. Нажмите **«Развернуть»**
5. Разрешите доступ если появится окно прав

---

## Шаг 6 — Скопировать ссылку

После развёртывания появится ссылка вида:
```
https://script.google.com/macros/s/XXXXXXXXXXXX/exec
```

**Отправьте эту ссылку команде в WhatsApp.**
Открывается в любом браузере без логина и регистрации.

---

## Обновление приложения

Если внесли изменения в index.html:
1. Откройте проект в script.google.com
2. Обновите содержимое файла index.html
3. «Развернуть» → **«Управление развертываниями»**
4. Нажмите ✏️ редактировать → **«Новая версия»** → Развернуть

---

⚠️ Примечание: у каждого пользователя свои данные (сохраняются в браузере).
Для общей базы данных нужно подключить Firebase (следующий шаг).
